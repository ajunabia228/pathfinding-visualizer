Welcome to the Pathfinding Visualizer

This app lets you explore how different pathfinding algorithms work on a grid.

  • Click or drag on the grid to place walls.
  • Green = Start node
  • Red = End node
  • Black = Wall
  • Blue = Visited node
  • Yellow = Final path
  • Select an algorithm and click Visualize.
  • You can pause and resume animations.
